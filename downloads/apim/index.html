Installation Steps

A detailed installation guide is available in the documentation.
If you want to start quickly with Gravitee.io API Management, we provide a Docker Compose to create a demo environment. Please have a look to this readme.

Gravitee.io API Gateway

Download

Download and unzip the latest Gravitee.io API Gateway distribution

Start

Run bin/gravitee on Unix
or bin/gravitee.bat on Windows

Run

Access the Gateway via http://localhost:8082/

Gravitee.io Management API

Download

Download and unzip the latest Gravitee.io Management API distribution

Start

Run bin/gravitee on Unix
or bin/gravitee.bat on Windows

Run

Access the Management API via http://localhost:8083/management

Gravitee.io Management UI - API Portal

Download

Download and unzip the latest Gravitee.io Management UI distribution

Deploy

Deploy the API Portal
on a HTTP server

Run

Access the API Portal via http://localhost
